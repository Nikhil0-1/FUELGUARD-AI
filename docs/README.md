# FuelGuard AI — Smart Fuel Accuracy & Monitoring Platform

FuelGuard AI is a complete commercial-grade IoT solution designed to monitor water/fuel delivery systems using the **YF-S201 Flow Sensor**, **ESP8266 NodeMCU**, **16x2 LCD display**, and a cloud-based **React & Firebase** web console.

## Project Structure
- [/firmware](file:///c:/Users/mdsha/Desktop/fuel/firmware/): Arduino C++ code for ESP8266 controller.
- [/web](file:///c:/Users/mdsha/Desktop/fuel/web/): React+Vite web dashboard console.
- [/firebase](file:///c:/Users/mdsha/Desktop/fuel/firebase/): Firebase rules configuration.
- [/docs](file:///c:/Users/mdsha/Desktop/fuel/docs/): Complete setup guides and wiring documentation.

## Hardware Components
- ESP8266 NodeMCU
- YF-S201 Flow Sensor
- 16x2 LCD with I2C Backpack
- 12V Power Supply + 12V to 5V Buck Converter
